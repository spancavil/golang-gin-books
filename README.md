
# Golang - Gin books management

A basic books CRUD (create read update and delete)
with Golang - Gin framework.

### Run Locally

Clone the project

```bash
  git clone https://github.com/spancavil/golang-gin-books.git
```

Go to the project directory

```bash
  cd my-project
```

Run the API

```bash
  go run .
```
